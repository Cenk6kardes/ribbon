 ISUP COT Test

Use this procedure to perform an ISUP (Integrated Services Digital Network User Part) continuity test (ICOT) on a single trunk member for a specific CLLI.

This test can only be performed on outgoing or 2-way ISUP trunks.

Prerequisites

The trunk must be in a maintenance busy (MB) state.

Start

  1. Launch the Trunk Maintenance Manager (TMM) GUI.
  2. Click Maintenance by Trunk CLLI and then ISUP COT Test.


  3. Select Gateway Name and Trunk CLLI
  4. Enter the member number.
  5. Select Run Test from the Panel.


If you encounter unexpected errors, such as "test failed," review the details and determine a solution, or contact your next level of support.
