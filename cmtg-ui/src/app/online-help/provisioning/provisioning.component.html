Functional Description

The Provisioning panel displays configuration data associated with the Gateway Controller. The name and unit IP addresses are displayed for the currently selected gateway controller at the top of the main panel. A status bar, located at the bottom of the main panel, displays operation messages. The previous twenty messages may be displayed via the dropdown list.

For a description of the tabbed panels contained in the Provisioning Panel, refer to Controller, Gateways, Lines, Carriers and QoS Collectors.
