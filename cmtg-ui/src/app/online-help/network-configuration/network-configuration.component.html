GR-834 GWs may be added and deleted by the buttons located under the GR-834 panel. Refer to the following procedures for information on these functions:

Add a GR-834 GW

Use this procedure to add a GR-834 GW.

  1. Display the Network Configuration panel.
  2. Select the GR-834 GWs tabbed panel, and click the "New GR-834 GWs" button to display the Add GR-834 GWs dialog.
  3. Enter the G-834 Name, Type, port, IP Address, User Name and Password.
  4. Click the SAVE button to confirm the request, or the Cancel button to abort the request.
  5. The GR-834 GW will appear in the list following successful addition.

Change an GR-834 GWs

Use this procedure to change Application Layer Gateway (ALG) server attributes.

  1. Display the Network Configuration panel.
  2. Select an GR-834 from the list and click the Edit icon.
  3. Enter your changes to the IP address, Port, username, password.
  4. Click the SAVE button to confirm the request, or the Cancel button to abort the request.
  5. GR-834 list is updated following a successful change.

Delete a GR-834 GW

Use this procedure to delete a GR-834 GW.

  1. Display the Network Configuration panel.
  2. Select the GR-834 GWs tabbed panel.
  3. Select a GR-834 GW from the list and click the Delete button.
  4. Click the Yes button to confirm the request, or the No button to abort the request.
  5. The GR-834 GW will be removed from the list following successful deletion.
