Functional Description

The Carriers panel provides access to configuration data for carriers associated with the selected gateway controller.

A common search interface is located at the top, while the lower section contains a table to display the retrieved records and functions specific to Carriers. The table supports single record selection for the Delete and Display functions. Fields that are not applicable for a record are blank.

Different versions of the Carriers panel are displayed based upon the C20 being configured for an International or North American deployment.

International Carriers Panel

Displays the Add Carrier dialog. Refer to the Add a Carrier procedure for more information.

Use this procedure to add a Carrier to a Gateway Controller.

  1. Display the Provisioning panel.
  2. Select the Carriers panel.
  3. Click the New Carrier button to display the Add Carrier dialog.
  4. Enter the carrier name and gateway name.
  5. Enter the optional Start terminal number value if desired.
  6. For PRI carriers, select the PRI / DPNSS checkbox and enter the IID value.
  7. For DPNSS carriers, select the PRI / DPNSS checkbox and enter the DUA Integer Interface Identifier as the IID. This should correspond to the value provisioned on the gateway for the carrier. The DUA IID value cannot be 0 and must be unique for a given gateway.
  8. For H.323 EPG carrier groups, select the H.323 checkbox and enter the required number of ports.
  9. For VoIP VPN EPG carrier groups, select the VoIP VPN checkbox and enter the required number of ports.
  10. Click the Add button to confirm the request, or the Cancel button to abort the request.

Displays a confirmation dialog for deletion of the selected carrier.

Use this procedure to delete a carrier from a Gateway Controller.

  1. Display the Provisioning panel.
  2. Select the Carriers panel.
  3. Display the carrier to be deleted in the Carrier List by either specifying the name, or other criteria, in the Retrieval criteria field then clicking the Retrieve button, or by clicking the Retrieve All button.
  4. Select the carrier from the Carrier List and click the Delete button which displays the Confirm Carrier Delete dialog.
  5. Click the Yes button to confirm the request or the No button to abort the request.

Display the trunks for the selected carrier.
